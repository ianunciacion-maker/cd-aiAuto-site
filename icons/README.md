# Icons Directory

This directory contains all icon assets for the Ai-Auto project.

## Structure

- `ui/` - User interface icons
- `social/` - Social media icons
- `brand/` - Brand-specific icons
- `flags/` - Country flags (if needed)

## Formats

- SVG format preferred for scalability
- Include multiple sizes for raster formats
- Use consistent stroke width and style
- Provide both filled and outline variants

## Usage

Icons should be optimized for:
- Small sizes (16px, 24px, 32px)
- High DPI displays
- Dark/light mode compatibility
- Accessibility (include aria-labels)

## Naming Convention

Use kebab-case with descriptive names:
- `icon-menu.svg`
- `icon-automation.svg`
- `social-twitter.svg`
