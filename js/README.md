# JavaScript Directory

This directory contains all JavaScript files for the Ai-Auto project.

## Structure

- `components/` - Reusable UI components (navigation, forms)
- `modules/` - Feature-specific modules (theme, animations)

## Main Files

- `main.js` - Main application entry point (ES6 module imports)
- `components/navigation.js` - SiteNavigation class for page navigation
- `components/tool-navigation.js` - ToolNavigation class for tool pages
- `components/forms.js` - FormHandler class for form validation
- `modules/theme.js` - ThemeManager class for dark mode
- `modules/animations.js` - AnimationController class for scroll animations
