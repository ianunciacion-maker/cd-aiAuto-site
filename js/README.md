# JavaScript Directory

This directory contains all JavaScript files for the Ai-Auto project.

## Structure

- `components/` - Reusable UI components
- `utils/` - Utility functions and helpers
- `modules/` - Feature-specific modules
- `vendor/` - Third-party libraries (if any)

## Main Files

- `main.js` - Main application entry point
- `navigation.js` - Navigation and menu functionality
- `theme.js` - Dark mode and theme switching
- `animations.js` - Scroll animations and interactions
- `forms.js` - Form handling and validation
