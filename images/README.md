# Images Directory

This directory contains all image assets for the Ai-Auto project.

## Structure

- `heroes/` - Hero section images
- `icons/` - Icon files and illustrations
- `logos/` - Logo variations
- `screenshots/` - Application screenshots
- `team/` - Team member photos
- `backgrounds/` - Background images and patterns

## Formats

- Use WebP format for better compression
- Provide fallbacks in PNG/JPEG
- Optimize all images for web
- Include responsive image sets where appropriate

## Naming Convention

Use kebab-case with descriptive names:
- `hero-illustration.webp`
- `team-clint-day.webp`
- `icon-automation.svg`
